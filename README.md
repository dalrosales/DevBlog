# dev.alerosa.me

Personal tech blog built with [Astro](https://astro.build) and the [MultiTerm](https://github.com/stelcodes/multiterm-astro) theme. Deployed automatically to [Azure Static Web Apps](https://azure.microsoft.com/en-us/products/app-service/static) via GitHub Actions on every push to `main`.

## Stack

- **Framework:** Astro
- **Theme:** MultiTerm
- **Hosting:** Azure Static Web Apps (free tier)
- **CI/CD:** GitHub Actions

## Local Development

```bash
npm install
npm run dev
```

Runs on `http://localhost:4321` by default.
